# Student Grade Tracker

A command-line grade tracking system built in Python that helps manage student grades and generate reports.

## Features
- Add students
- Add grades per subject
- View individual student report with average and letter grade
- View all students and their averages
- Delete students
- Data saved persistently using JSON

## How to Run
```bash
python3 grades.py
```

## Tech Stack
- Language: Python
- Concepts: File I/O, JSON, Functions, Dictionaries
